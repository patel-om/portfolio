VeriTriage - Verification Intelligence Platform

Automated Regression Failure Triage

Built an open-source Python platform that turns verification artifacts (UVM/VCS/Questa/Xcelium logs, compile logs, coverage, test metadata) into a normalized Evidence Graph, then reasons over it like a DV engineer: a multi-stage pipeline ranks competing hypotheses (RTL bug vs testbench vs infrastructure) with fully traceable confidence and categorized debug steps. Every analysis is recorded into a persistent regression database with deterministic failure signatures and similarity search, so the platform answers "have we seen this before?", surfaces historical root causes, and renders team-level analytics (failure clusters, module hotspots, trends) as an engineering dashboard. The optional AI layer reviews only selected evidence and cites node IDs, never raw files.
